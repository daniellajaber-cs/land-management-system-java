# Land Management System (Java OOP)

This project is a Java-based Object-Oriented Programming system developed as a university OOP project.

## Features
- Add agriculture and construction lands
- Delete and search land by ID
- Display all lands
- Count land types
- Update land information
- Merge two lands
- Split land by percentage
- Calculate land price
- File handling for storing and reading land data

## Concepts Used
- Java
- Object-Oriented Programming (Inheritance, Polymorphism)
- Arrays
- File Handling

## How to Run
1. Open the project in a Java IDE (Eclipse recommended)
2. Run `Management_Menu.java`
